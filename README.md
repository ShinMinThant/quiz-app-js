# 🧠 Quiz App (JavaScript)

A simple and interactive Quiz Application built using **HTML, CSS, and Vanilla JavaScript**.  
Users can answer multiple-choice questions, receive instant feedback, and see their final score.

This project was created to practice JavaScript fundamentals such as DOM manipulation, event handling, arrays, objects, and application logic.

## ✨ Features

- ✅ Multiple-choice quiz questions
- ✅ Select answers interactively
- ✅ Instant answer feedback
- ✅ Next question navigation
- ✅ Score calculation
- ✅ Final result display
- ✅ Responsive design for different screen sizes

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
